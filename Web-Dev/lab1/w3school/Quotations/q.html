Using HTML Quotations Elements

The <q> element is used for short inline quotations. It typically adds quotation marks around the quoted text.

Example of q:

She said, Learning never exhausts the mind. - Leonardo da Vinci

The <abbr> element is used to define an abbreviation or acronym, providing a full description when hovered over.

Example of abbr:

The <abbr title="HyperText Markup Language">HTML</abbr> is the standard markup language for creating web pages.

The <address> element is used to provide contact information for the author or owner of a document or article.

Example of address:

Contact us at:
Email:blalba@gmail Phone: +123-456-7890
123 Learning St., Knowledge City, Edu Country
