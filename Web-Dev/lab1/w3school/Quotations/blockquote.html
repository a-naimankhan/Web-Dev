Using the <blockquote> Element

The <blockquote> element is used to indicate that the enclosed text is an extended quotation. It is typically displayed as an indented block of text.

Example of Blockquote:

"The only limit to our realization of tomorrow is our doubts of today."
- Franklin D. Roosevelt

In this example, the quote from Franklin D. Roosevelt is enclosed within the <blockquote> tags, which will render it as a distinct block of text, usually indented from the rest of the content.
