HTML Formatting Tags

HTML provides several tags for formatting text. Here are some of the most commonly used formatting tags:

  • <b>...</b>: Bold text
  • <i>...</i>: Italic text
  • <u>...</u>: Underlined text
  • <s>...</s>: Strikethrough text
  • <mark>...</mark>: Highlighted text
  • <small>...</small>: Smaller text
  • <sub>...</sub>: Subscript text
  • <sup>...</sup>: Superscript text

Example usage:

This is bold text.

This is italic text.

This is underlined text.

This is strikethrough text.

This is highlighted text.

This is smaller text.

This is subscript: H2O.

This is superscript: E = mc2.
